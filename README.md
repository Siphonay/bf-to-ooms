# bf-to-ooms

Traducteur de brainfuck en [ouai-ouai-miaou-super](https://github.com/Siphonay/ouai-ouai-miaou-super)

## compilation

Il va te falloir [Crystal](https://crystal-lang.org/) (oui désolé c’est pas le même langage que l’interpréteur j’avais envie de faire les deux), ensuite c’est facille tu clones ce dépôt avec `git clone https://github.com/Siphonay/bf-to-ooms` et ensuite tu le compile avec `shards build`

## utilisation

```
usage: bf-to-ooms [-e expression] [file path]
```
